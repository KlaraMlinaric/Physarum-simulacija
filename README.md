Na temelju prikupljenih podataka i novih saznanja o kretanju vrste Physarum P. napravljen je program u programskom jeziku Python,
koji simulira njegovo kretanje. Program se nalazi u obliku .exe datoteke pa je za njegov rad potrebno samo skinuti datoteku i otvoriti je na računalu. 
Korisnik programa svojevoljno bira položaj hrane, a mjenjanjem koda može postavljati različite labirintske uvijete. 
Program je namijenjen svim zainteresiranim osobama koje žele ponoviti ovo istraživanje ali nisu u mogućnosti nabaviti Physarum P.
